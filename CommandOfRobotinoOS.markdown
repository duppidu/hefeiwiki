## RobotinoOS   
### Kommandos
<a href="#Ankername">Link</a>
BIOS-Setup 
>F2 oder Del 

Bootmenue 
>F7

Grafische Oberfläche beenden 
>Ctrl + Alt + Backspace

Java Version überprüfen
>java -version

> * sudo dpkg-reconfigure keyboard-configuration
>

>
